Cancer Hallmarks Analytics Tool API

Querying

You can query CHAT using the following URL:

http://chat.lionproject.net/chartdata?q=<query>&measure=<measure>;&hallmarks=<hallmarks>

where:
  • <query> is the query search string, the search string can include boolean combinations using "AND" and "OR"
  • <measure> can either be: "count", "cprop", "pmi", "npmi"
  • <hallmarks> can either be: "top", to indicate that the returned results is for the original hallmarks of cancer, or "full" for the entire taxonomy
The following is an example for the query "p53", using the "npmi" measure for the entire taxonomy:

http://chat.lionproject.net/chartdata?q=p53&measure=npmi&hallmarks=full

Document texts

You can get the original text for a given abstract by its PubMed Identifier using the following:

http://chat.lionproject.net/pubmed/<ID>/text

For example: http://chat.lionproject.net/pubmed/11574421/text

Annotations

You can get the annotation for a given abstract by its PubMed Identifier using the following:

http://chat.lionproject.net/pubmed/<ID>/annotations

For example: http://chat.lionproject.net/pubmed/11574421/annotations

This would return the annotations in JSON format where each annotation span contains the following:

  • body: contains the code for the hallmark annotation
  • id: the annotation span id in the file
  • target: the starting and ending character position of the span, as they occur in the document text (see the previous section for retrieving the text).
